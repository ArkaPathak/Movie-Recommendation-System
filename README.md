# ReelMatch Movie Recommendation Chatbot

ReelMatch is a Streamlit-based movie recommendation chatbot that suggests movies based on the user’s mood, genre preference, and priorities. It uses Gemini 3.5 Flash through the Google GenAI SDK.

## Features

- Clean and modern Streamlit UI
- Chatbot-style conversation
- Personalized movie suggestions using user mood and priorities
- Header and footer branding
- Reset conversation support

## Project Structure

- `app.py` — Streamlit frontend
- `ChatBot.py` — Gemini-powered recommendation logic
- `.env` — API key and model configuration
- `requirements.txt` — Python dependencies

## Tech Stack

- Python
- Streamlit
- Google GenAI SDK
- python-dotenv
